SummarAI — Instant Web Summarizer

An AI-powered web summarizer that extracts key points from any URL in seconds. Built with Flask and OpenRouter.

Features

* Paste any URL and get an instant summary
* Extracts key points and conclusions
* Clean dark UI with copy to clipboard
* Fully responsive on mobile and desktop

Tech Stack

* Python + Flask
* OpenRouter API (free)
* BeautifulSoup4 (web scraping)
* Vanilla HTML/CSS/JS
